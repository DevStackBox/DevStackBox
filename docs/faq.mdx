## What is DevStackBox?

DevStackBox is a free, open-source local PHP development stack for Windows. It bundles Apache, MySQL, PHP, and phpMyAdmin in a single desktop app with a modern UI.

## Is DevStackBox free?

Yes. DevStackBox is MIT-licensed open source. There is no paid tier.

## Where does DevStackBox install?

The application installs to `C:\devstackbox` by default (you can choose another path). User data - web projects, databases, configs, and logs - is stored in `%LOCALAPPDATA%\DevStackBox\`.

## Does DevStackBox collect telemetry?

No. DevStackBox does not collect usage data or phone home. See the [Privacy Policy](/privacy).

## How do I report a bug?

Open a [GitHub Issue](https://github.com/DevStackBox/DevStackBox/issues) or start a [Discussion](https://github.com/DevStackBox/DevStackBox/discussions).

## How do I report a security issue?

Email **hello@devstackbox.com**. Do not open public issues for vulnerabilities. See [Security](/security).
